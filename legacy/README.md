# Fichiers legacy TP1 / TP2

Ce dossier contient les sources des étapes intermédiaires des TP 1 et 2,
conservées pour référence. Ces fichiers ne sont PAS compilés par le Makefile
principal et ne font pas partie du rendu final du TP3.

- `servudp.c` / `cliudp.c` : serveur/client UDP de base (TP2 étape 1)
- `servbeuip.c` / `clibeuip.c` : serveur/client BEUIP autonomes (TP2 étapes 2-3)
- `test_creme.c` : programme de test autonome de la librairie creme
